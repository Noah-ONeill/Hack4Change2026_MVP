# Hack4Change2026_MVP
An application to help homeless organizations to coordinate with people who want to donate items.
The app will allow the basic user to make posts of the items they have, and for the shelter staff to post items they need.
The items then get matched through the coordinators and once items are recieved shelters can then update their stock.

AI Usage:
- Used ChatGPT to brainstorm and help come up with general system architecture
- Used Claude to assist with a basic 1st iteration of the application
